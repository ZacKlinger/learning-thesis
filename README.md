# learning-thesis

Working repo for an essay arguing the optimistic case for AI/tech in education — specifically that it should enable ambitious learning that wasn't previously possible, while protecting teacher agency.

Mastery first, essay second. The bulk of this repo is the research substrate: a layered canon of learning sciences literature, a claim-evidence ledger, and a weekly digest of advances.

See `CLAUDE.md` for the protocol the research agent follows.
